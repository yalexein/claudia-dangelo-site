# Final Chirp Prompt

Use this only after `bird-idle.webp` has been approved.

## Paste-ready prompt

Use the approved `bird-idle.webp` as a reference image.

```text
Use case: photorealistic-natural
Asset type: interactive website sprite frame with transparent background
Input images: Image 1: approved bird-idle.webp reference for species, crop, scale, lighting, and silhouette consistency
Primary request: create a chirping variation of the same photorealistic white wagtail / ballerina bianca with only a very small natural vocal pose change
Subject: the same full-body white wagtail, same side profile, beak opened slightly for a short chirp, tiny chest lift, elegant alert posture, both feet visible, tail fully visible
Style/medium: photorealistic editorial wildlife photography, refined and natural
Scene/backdrop: transparent background only, no branch, no ground, no environment, no prop
Composition/framing: match the reference image as closely as possible, same crop, same scale, same lens feel, same silhouette family, same centered placement
Lighting/mood: identical soft daylight from upper left, same realistic feather shadows, same contrast level, no dramatic relighting
Color palette: same pearl white plumage, cool grey back and wings, charcoal-black cap and bib
Materials/textures: same feather realism, same clean cutout edges, same delicate legs, same natural beak texture
Constraints: change only the minimum needed to suggest a short natural chirp; preserve species markings, anatomy, scale, crop, light direction, and transparency; no text; no watermark
Avoid: wide-open beak, screaming pose, cartoon expression, wing spread, different camera angle, different scale, dirty transparency, added objects
```

## Tightened variant

Use this if the model exaggerates the chirp:

```text
Keep the bird almost identical to the reference idle frame. Change only the beak to a small natural opening and lift the chest very slightly. Keep feet aligned, tail aligned, crop identical, scale identical, lighting identical, transparent background, no stylization.
```

## Recovery prompts

### If the beak opens too much

```text
Reduce the beak opening so it reads as a tiny natural chirp, not a strong call or exaggerated vocal pose.
```

### If the body changes too much

```text
Return the body, tail, and feet much closer to the reference idle frame. Change only the beak and a very subtle chest lift.
```

### If it becomes too expressive

```text
Make the pose calmer and more realistic. Keep the bird elegant, restrained, and natural, with no cartoon emotion.
```

## Approval checklist

- reads as the same exact bird as `bird-idle.webp`
- beak is only slightly open
- body change is minimal
- feet alignment is stable
- tail alignment is stable
- crop and scale match the idle frame
- transparent cutout stays clean
