# Immagini progetto

Questa cartella raccoglie gli asset sorgente che useremo per il sito.

Struttura consigliata:

- `landing/`
- `bio/`
- `collage/`
- `scritture/`

Per la landing, ogni oggetto interattivo puo avere una sottocartella dedicata:

- `landing/desk/`
- `landing/window/`
- `landing/poster/`
- `landing/photo/`

Esempio nomi file:

- `desk-default.png`
- `desk-drawer-open.png`
- `poster-default.png`
- `poster-hover.png`

Nota:
- qui possiamo tenere i file originali di lavoro;
- gli asset effettivamente serviti dal sito vanno poi copiati in `public/images/`.
